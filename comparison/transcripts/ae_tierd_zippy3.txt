fixture: dir1 = 27 entries, dir2 = 3, max dir = 2). Search token `ART`

Notes (server bytes, with the adapter's input echo removed):
- 26137 `\b\n` + 26172 `\b\n` => the two CRLF blanks before the first
  `Scanning directory N` header on the immediate path (no prompt between).
- Out-of-range: `getDirSpan` prints `\b\nNo such directory.\b\n\b\n`
  (express.e:26905); with the 26137 blank that is two leading CRLFs, the
  text, then two trailing CRLFs (a third before the prompt is the menu).
- The trailing `<MENU PROMPT>` is the FS-UAE menu loop's expert-mode
  prompt; NextExpress re-renders the menu screen instead (slice-
  independent, see COMMAND_PARITY.md).
